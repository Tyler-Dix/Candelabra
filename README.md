# Candelabra

A text-based adventure game that is "tile-based"

Features:
1. Can move, look, attack, defend, run, examine, etc.
2. Two "floors"
3. Interaction with non-combative NPCs (dialogue)
4. A focus on the "Candelabra"
5. Equip item(s)
6. Combat with entities
